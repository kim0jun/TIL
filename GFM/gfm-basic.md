GFM Basic
=================
본 문서는 kim0jun이 gfm을 공부하기위하여 작성한 문서입니다.  

##GFM이란?

GFM은 표준 Markdown(SM)에 조금더 편하게 변경한 github 버젼입니다. 
그리고 Markdown은 Plain text로 작성된 파일을 HTML로 변경해서 보여주기 위한 문법입니다.

##GFM문법

###헤더
- h1 = 또는 #
#H1
- H2 – 또는 ##
##H2
- H3 ### 
###3
- H4 #### 
####4
- H5 #####
#####5
- H6 ######
######6

###줄바꿈
문장의 줄바꿈을 그대로 반영합니다(Enter)

###들여쓰기
> >,>>로 표현이가능합니다.
>> >,>>로 표현이가능합니다.
